# Git Command Finder

A single-purpose React web application to help developers quickly discover and copy Git commands.

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling with terminal-inspired dark theme
- **Lucide React** - Icon library
- **Vitest** - Testing framework
- **React Testing Library** - Component testing
- **fast-check** - Property-based testing

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Test

```bash
npm test
```

### Preview Production Build

```bash
npm run preview
```

## Theme

The application uses a terminal-inspired aesthetic:
- Background: Black (#000000)
- Text: Green (#00ff00)
- Monospace fonts for code snippets
